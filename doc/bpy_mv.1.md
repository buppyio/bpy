# NAME

bpy put - move a file or folder

# SYNOPSIS

The mv command lets you move a file or folder from one path to another within a ref

# Usage

```bpy mv [-ref=default] src dest```

# Example

Rename a file

```
$ bpy ls
hello.txt
$ bpy mv hello.txt renamed.txt
$ bpy ls
renamed.txt
```
