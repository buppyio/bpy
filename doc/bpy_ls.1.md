# NAME

bpy ls - list directory contents

# SYNOPSIS

The ls command lets you list the contents of a bpy directory in a similar
way that the system ls command does on unixy systems.

# Usage

```bpy ls [-at=TIMESPEC] [path]```

# Example

List a directory

```
$ bpy ls /hello/
hello.txt
$ bpy ls
hello/
```
