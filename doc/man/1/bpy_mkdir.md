# Name

bpy mkdir - create a directory in the bpy drive.

# Synopsis

The mkdir command allows the creation of a new folder.

# Usage

```$ bpy mkdir path```

# Example

make a directory:

```
$ bpy ls
$ bpy mkdir files
$ bpy ls
files/
```
