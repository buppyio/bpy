# Name

bpy zip - export a bpy folder to a zip archive

# Synopsis

The zip command creates a zip archive out of the specified folder writing the output to stdout.

# Usage

```bpy zip [-at=TIMESPEC] src```

# Example

Fetch the entire root as a zip archive:

```
$ bpy zip / > out.zip
```
