# NAME

bpy zip - export a bpy folder to a zip archive

# SYNOPSIS

The tar command creates a tar archive out of the specified ref and folder.

# Usage

```bpy tar [-ref=default] src```

# Example

Fetch the entire ref as a zip archive

```
$ bpy zip / > out.zip
```
