# NAME

bpy mkdir - create a directory in the bpy filesystem

# SYNOPSIS

The mkdir command allows the creation of a new folder in the specified ref.

# Usage

```$ bpy mkdir [-ref=default] path```

# Example

Remove a file

```
$ bpy ls
$ bpy mkdir files
$ bpy ls
files/
```
